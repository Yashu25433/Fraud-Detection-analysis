# Fraud Detection Analysis Using AI

This project demonstrates an end-to-end machine learning workflow for fraud detection using Python.

## Features
- Synthetic fraud dataset creation
- Data preprocessing & scaling
- Class imbalance handling
- Multiple ML models (LR, RF, GradientBoosting)
- Model evaluation using Precision-Recall AUC

## How to Run
Install dependencies:
```
pip install -r requirements.txt
```

Run script:
```
python fraud_detection.py
```
